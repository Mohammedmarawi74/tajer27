# 🎨 ميزة اختيار الشعار

## نظرة عامة
تم إضافة نظام متقدم لاختيار الشعار يتيح للمستخدمين:
- الاختيار من بين 4 شعارات جاهزة
- رفع شعار مخصص
- إزالة الشعار تماماً

## 📁 موقع الشعارات
الشعارات موجودة في المجلد:
```
public/logooo/
├── logo-1.png
├── logo-2.png
├── logo-3.png
└── logo-4.png
```

## 🎯 كيفية الاستخدام

### 1. اختيار شعار جاهز
1. افتح لوحة التحكم (EditorSidebar)
2. اذهب إلى تبويب **التصميم**
3. في قسم **شعار المنصة**، سترى 4 شعارات جاهزة
4. انقر على الشعار الذي تريده
5. سيتم تحديده فوراً مع علامة ✓ خضراء

### 2. رفع شعار مخصص
1. في نفس القسم، اضغط على **رفع شعار خاص**
2. اختر ملف الصورة من جهازك
3. سيتم استخدامه تلقائياً

### 3. إزالة الشعار
1. عندما يكون هناك شعار محدد
2. سيظهر زر أحمر **إزالة الشعار تماماً**
3. اضغط عليه لإزالة الشعار من التصميم

## 📍 مكان الشعار في التصميم
- **الموقع**: الزاوية العلوية اليمنى من الشريحة
- **الحجم**: 24x24 بكسل (6rem في CSS)
- **النص المصاحب**: "Radar Al-Mustathmir"
- **الخلفية**: دائرة بلون أخضر شفاف (emerald-500/20)

## 🎨 التنسيقات CSS

### الكلاسات المستخدمة:
```css
.slide-logo-section      /* حاوية الشعار */
.slide-logo-text         /* النص بجانب الشعار */
.slide-logo-image        /* صورة الشعار */
```

### الأبعاد:
```css
.slide-logo-image {
  width: 1.5rem;   /* 24px */
  height: 1.5rem;  /* 24px */
}
```

## 💡 إضافة شعارات جديدة

لإضافة شعارات جديدة:

1. أضف ملف الصورة إلى `public/logooo/`
2. افتح `constants.ts`
3. أضف الشعار الجديد إلى مصفوفة `AVAILABLE_LOGOS`:

```typescript
export const AVAILABLE_LOGOS = [
  // ... الشعارات الحالية
  { id: 'logo-5', name: 'الشعار الخامس', path: '/logooo/logo-5.png' },
];
```

## 🔧 التعديل على التصميم

### تغيير حجم الشعار:
عدل في `styles.css`:
```css
.slide-logo-image {
  width: 2rem;   /* حجم جديد */
  height: 2rem;  /* حجم جديد */
}
```

### تغيير موقع الشعار:
عدل في `styles.css`:
```css
.slide-logo-section {
  top: 3rem;     /* مسافة من الأعلى */
  right: 3rem;   /* مسافة من اليمين */
}
```
